AI Face Interview with Emotion Detection

Smart Interview System using AI & Machine Learning

📌 Project Overview

This project is an AI-based interview system that analyzes candidate emotions using facial expressions during an interview. It helps in understanding confidence, stress, and engagement levels.

🛠️ Technologies Used

  • Python
  • OpenCV
  • Machine Learning
  • MySQL

⚙️ Features

  • Real-time face detection
  • Emotion recognition
  • Automated interview analysis
  • User-friendly interface
